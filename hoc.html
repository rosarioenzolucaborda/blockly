Using Blockly for the Hour of Code

To run Blockly on a computer when not connected to the Internet:
  1. Download blockly.zip while connected to the Internet.
  2. Transfer the file to any other computers and unzip it.
  3. Open the file "blockly-zip/index.html" by either double-clicking on it or through File->Open File in a modern web browser.
  4. Select your preferred language from the dropdown menu at the top right of the screen.
Languages supported: العربية, Brezhoneg, Català, Česky, Dansk, Deutsch, Ελληνικά, Español, Euskara, فارسی, Français, Galego, Interlingua, Magyar, Íslenska, Italiano, 日本語, 한국어, Latviešu, Македонски, Bahasa Melayu, Nederlands, Polski, Piemontèis, Română, Português Brasileiro, Русский, Slovenčina, Српски, Kishwahili, Svenska, ภาษาไทย, Türkçe, Українська, Tiếng Việt, 簡體中文, 正體中文.
